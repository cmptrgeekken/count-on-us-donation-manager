# DPA Request Page Draft

This draft is a working document for Count On Us.

It is not legal advice and should be reviewed by qualified counsel before publication or use.

## Request a Data Processing Addendum

If your business requires a Data Processing Addendum ("DPA") for Count On Us, you may request our standard DPA using the contact path listed below.

## What to Include

Please include the following information in your request:

- merchant legal entity name
- Shopify store domain
- primary contact name
- contact email
- signing contact name and title, if different
- any internal procurement or privacy ticket/reference number you want us to include

## Where to Send the Request

Suggested first-pass publication model:

- email: `privacy@your-domain.example`
- or a public compliance/contact form hosted at `/dpa`

## What Happens Next

After we receive the request, we will:

1. confirm the requesting merchant and store
2. request any missing information needed to prepare the paperwork
3. provide our standard DPA for review
4. continue the signature process through the agreed contact path

## Important Notes

- A DPA request path is not the same thing as uninstall or data deletion.
- The DPA is part of the legal/data-processing relationship while the app is being used.
- Uninstall and deletion behavior should still be described separately in the privacy policy and implemented through the app's operational deletion workflow.
